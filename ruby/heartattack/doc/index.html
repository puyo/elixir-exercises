                                                          Introduction

                                                          A love simulation game...

                                                          Influences

                                                          Among its influences, this game has the following games and anime: Rockstar, Princess Maker, Megatokyo, Himechan no Ribon, Kareshi Kanajo no Jijou.

                                                          Character Creation
                                                          1. Select sex.
                                                          2. Enter name.
                                                          3. Select birthday. Your Horoscope will be given to you based on your birthday.
                                                          4. Spend points on Primary Attributes. These will be fixed for the duration of the game.
                                                          Primary Attributes

                                                          Metabolism, Beauty and Intelligence.

                                                          Horoscope

                                                          One of Aries, Aquarius, Cancer, Capricorn, Gemini, Leo, Libra, Pisces, Sagitarius, Scorpio, Taurus, Virgo.

                                                          The fortune telling machine at the arcade can give you good or bad readings for you and a friend based on your horoscope.

                                                          Playing the Game
                                                          Goal

                                                          The goal of the game is to aquire a boyfriend or girlfriend.

                                                          Basic Gameplay

                                                          There are two stages to each day, morning and afternoon. Each stage is 6 hours long. For each, you first select one of a set of possible Locations where it will take place (travel time is negligible) and then select Activities from a menu until the stage's time is up.

                                                          If an activity takes longer than can be fitted in the morning, a warning will be shown, and if you wish, the extra time will be deducted from the afternoon.

                                                          When morning time is used up, afternoon arrives. In the afternoon the process is repeated. If you do something which takes too much time in the afternoon, a warning will be shown, and if you wish, you'll stay up late. The number of hours you stay up late are deducted from your Health.

                                                          'Hours left' (in the current stage) is also stored somewhere.

                                                          Schedule

                                                          To keep organised, you have a schedule feature in the game which simply keeps track of what invitations you've extended to whom and for when. When you Invite somebody, it fixes your activity for this time. When the time comes, you will simply receive a prompt asking if you'd like to proceed with the promise. If you wish, you can break your promise and do something else, but it costs a lot of friendship.

                                                          Sometimes other people will arrange parties or other events, and invite you. If you accept, the scheule works in the same way.

                                                          Store: Hash[Time => Array[(Friend, Location)]].

                                                          Activities

                                                          Activities are associated with People (e.g. Talk) and Items (e.g. Read). Each stage, the activity menu is built from all the activities provided by these things. When you select an activity from the menu, all the things which provide that activity are listed (e.g. if you select Chat, you can choose from all the people you can Chat to).

                                                          Stereotypes
                                                          Adjectives

                                                          Stereotypes can be prefixed with an adjective depending on your Primary Attributes.

                                                          The prefixes are: 'Buff', 'Babelicious' and 'Smart', which you get if one of your attributes Health, Style or Studiousness is greater than 70. Whichever is greatest will determine your prefix.

                                                          Stereotype Listing
                                                          Name Notes
                                                          Bimbo Aquired if you spend too much time reading Dolly or at the Mall.
                                                          Bitch Aquired if you spread bad gossip and insult people.
                                                          Class Clown Aquired if you act up and tell one too many jokes in class.
                                                          Money Bags Aquired if you always pay for people when you invite them places, or when they ask.
                                                          Mummy's Boy/Daddy's Girl Aquired if you spend too much time with your parents and refuse too many invitations.
                                                          Nerd Aquired if you spend too much time studying.
                                                          Pervert Aquired if you spend too much time looking at people who aren't even your friends, especially at the gym.
                                                          Rebel Aquired if you wag school and talk in class.
                                                          Runaround Aquired if you're ever caught with two different dates, or if you spread your lovin' around too much.
                                                          Scrooge Aquired if you don't pay for people when you invite them places or when they ask.
                                                          Teacher's Pet Aquired if you suck up to the teacher too much.
                                                          Workaholic Aquired if you're always missing invitations because you're working.
                                                          Random Events
                                                          Someone (not a friend) hits on your bf/gf. The new black changes colour (read Dolly to keep up!). Your favourite band breaks up (lose Style/happiness?).
                                                          Class Implementation
                                                          Person
                                                          Attribute Type Range Notes
                                                          Sex Enumeration M, F M = male, F = female.
                                                          Birthday Date Any Determines age and horoscope. Age differences can cause social stigmas.
                                                          Height String little, short, normal, tall, imposing
                                                          Weight String skinny, slim, normal, plump, fat
                                                          Hair String {HAIR_ADJ} {HAIR_COL}
                                                          Eyes String {EYES_ADJ} {EYES_COL}
                                                          Inventory Array[Item] 0..n
                                                          Relationships Hash[Person=>Relationship] 0..n See Relationship.
                                                          Student

                                                          Inherits Person.

                                                          Attribute Type Range Notes
                                                          Metabolism Integer 1-10 People with high metabolism don't get fat no matter what they eat. They seem fit and healthy.
                                                          Beauty Integer 1-10 Beautiful people attract more romantic interest.
                                                          Intelligence Integer 1-10 Intelligent people get good Grades and attract other intelligent people.
                                                          Health Integer 0-100 Health suffers if you eat poorly, stay up late, etc. If it reaches 0, your concerned parents ship you off to a fat farm (Game Over). Exercise and healthy food increase your health. Being fit and healthy can attract some people.
                                                          Style Integer 0-100 This represents your dress sense. Go shopping with a high Style rating, and you'll come back with super classy outfits. If you don't have any style, you may have to go shopping with a friend who has a high Style rating, and listen to their advice. Reading fashion magazines increases Style.
                                                          Extroversion Integer 0-100 Extroversion suffers when you don't talk to people enough, or go out with friends in groups. Some people like extroverts and some like introverts.
                                                          You

                                                          Inherits Student.

                                                          Attribute Type Range Notes
                                                          Gossip Integer 0-100 The amount of gossip you have accumulated. Increases when you gossip on the phone and observe friends. Increases the rewards of gossiping.
                                                          ParentalApproval Integer 0-100 0 = "the evil child", 100 = "the perfect angel". The happier your parents are with you, the more of their income they'll pass on to you as pocket money. If your parents aren't happy with you, they'll ground you or, if you really neglect them, ship you off to boarding school (Game Over).
                                                          TeacherApproval Integer 0-100
                                                          Job Enumeration TODO A part time job will reel in the money for you to spend on your loved ones. But you may be asked to work at inconvenient times. And, of course, if you don't turn up to your job repeatedly, you'll get the sack.
                                                          Promotions Integer 0-10 0 = new employee, 10 = inheriting the business. Your level of promotion. This combined with your job determines your salary.
                                                          BossApproval Integer 0-100 Each time your boss' approval hits 100, you get promoted, and it starts again at 20 + Job Level. If it hits 0 you're fired. Each time you miss work you lose Boss Approval.
                                                          Money Integer 0..n With money you can buy your loved ones presents, and take them to the cinema or the arcade. You'll have to be on good terms with your (wealthy) parents, or have a part-time job to get money though.
                                                          Friend

                                                          Inherits Student.

                                                          Attribute Type Range Notes
                                                          Sexuality Enumeration M, F, B M = like males, F = like females, B = likes both.
                                                          Resolve Integer 0-100 How hard this person's resolve is. Stubborn friends can go without Attention for longer and still maintain Friendship and Romance, however they are harder to befriend in the first place.
                                                          Likes[Metabolism] Integer 0-100
                                                          Likes[Beauty] Integer 0-100
                                                          Likes[Intelligence] Integer 0-100
                                                          Likes[Extroversion] Integer 0-100 How much does this person like extroverted people. Lower means they prefer introverted people. 50 means they have no preference. People *usually* like the opposite personality types as themselves.
                                                          Likes[Money] Integer 0-100 How important is material wealth to this person.
                                                          Adult

                                                          Inherits Person.

                                                          Relationship

                                                          Had between Persons. Inherits Integer. The value of the relationship represents the strength of your relationship with this person. 0 means you don't know this person. 50 means you consider this person your friend and 100 is bestest friends ever.

                                                          FriendRelationship

                                                          Had between Friends. Inherits Relationship.

                                                          Attribute Type Range Notes
                                                          Romance Integer 0-100 What proportion of the relationship is romantic. 100 means 100% of the strength of the relationship is romantic.
                                                          Attention Integer 0-100 How much attention they've had from you. Paying people too much attention when you're not yet their friend or romantic interest can get you labelled as a Pervert. Aside from this danger, attention is a good thing. Slowly declines over time. If it hits 0, your Friendship and Romance start dropping.
                                                          Item

                                                          Possessed by Persons or exist at Locations.

                                                          Attribute Type Range Notes
                                                          Owners Array[Person] 0..n Giving presents back is never a good idea. This is also used to determine whether or not you own something, so that you can't use things you haven't bought yet at the Mall, for example.
                                                          Cost Integer 1..n In dollars.
                                                          Categories Array[Enumeration] 1..n[Food, Jewellry, Stupid, Quiet, Athletic, Sexual, etc.] The type of an item is represented by the categories it fits into. For example, it's possible to have Stupid Jewellry (Phantom Keyring with Secret Compartment) or Sexual Food (body chocolate).
                                                          Description String Any string.
                                                          Effects Array[Effect] 0..n Using an item can have various effects on you.
                                                          Location

                                                          Has Persons and provides Activities for Persons.

                                                          Attribute Type Range Notes
                                                          Description String Any
                                                          Picture Image GIF, JPEG images
                                                          Items Array[Item] 0..n Things at this location. e.g. The library has textbooks and Dolly magazines. Shops at the Mall have items.
                                                          Activities Array[Activity] 0..n
                                                          Activity

                                                          Done by Persons at Locations.

                                                          Attribute Type Range Notes
                                                          Description String Any
                                                          Callback Script Set of prewritten activity scripts
